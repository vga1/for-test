## Огляд

Цей проект — це веб-додаток, який використовує PHP, Nginx та MySQL для управління даними. Додаток дозволяє виконувати запити до бази даних, а також надає простий інтерфейс для взаємодії з даними.

## Вимоги

- PHP
- React
- Nginx
